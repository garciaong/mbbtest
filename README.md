# How to run the application
* Go to thirdparty directory, execute third party application simulation (after succesful maven build)
```
> java -jar ./target/thirdparty.jar
```
* After thirdparty application is up and running, execute main application (after succesful maven build)
```
> java -jar ./target/main.jar
```
* The main application log file is available at /main/logs/application.log
